Naufal FzFr

Editor & Fotografer

Assalamu’alaikum. Saya Naufal, atau nama kerennya “Naufal FzFr”. Saya lahir di Jakarta pada tanggal 28 Maret 2003. Saat ini, saya sedang menempuh pendidikan di program studi Teknik Informatika. Meskipun saya memiliki latar belakang editing seperti foto dan video, saya memutuskan untuk mengejar karir di dunia TI yang penuh dengan koding-kodingan. Meskipun terbilang sulit, saya berusaha untuk menjadikan pengalaman ini sebagai tambahan skill baru.

Selain itu, saya juga pernah berkarya dalam bidang membuat short movie, edit foto dan video, mendesain, seni lukis kaligrafi arab, dan lain-lain. Terkadang, saya juga mendapat panggilan untuk menjadi seorang fotografer atau kameramen di acara-acara tertentu. Saya juga memiliki channel YouTube dengan nama “Naufal FzFr”. Meskipun awalnya bercita-cita menjadi seorang masinis, saya malah terjun ke dunia editing dan ditambah dengan dunia TI.

Meskipun terkadang cukup pusing dan menggila, saya yakin bahwa jika kita menjalani dengan ikhlas dan tawakal, maka akan mendapatkan berkah. Meskipun saya mempunyai karya yang belum sesuai ekspektasi, saya akan terus berlatih agar menjadi lebih baik lagi. Saya memohon do'a dari kawan-kawan agar saya bisa berkarir sukses dan membahagiakan kedua orang tua. Semoga semua keinginan kita tercapai. Salah satunya "semoga suatu saat nanti saya bisa berkunjung ke Arab Saudi dan Jepang". Terima kasih, wassalamu’alaikum.
